#### install:
```bash
npm install --save --save-exact @magic-themes/project
```

#### usage:

##### import:
```javascript
// config.mjs:

export default {
  //... other config
  THEME: '@magic-themes/project',

  // multiple themes
  // THEME: [...other themes, '@magic-themes/project'],
}
```

@magic will then import and use the theme automagically.
